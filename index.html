¡Bienvenidos al módulo de Lenguajes de Marcas y Sistemas de Gestión de la Información del ciclo de Desarrollo de Aplicaciones Multiplataforma!

Este curso está diseñado para que adquieras las habilidades necesarias en el manejo de lenguajes de marcas y sistemas de gestión de información, los cuales son fundamentales en el desarrollo de aplicaciones web modernas. A lo largo del módulo, aprenderás cómo estructurar, transmitir y presentar información utilizando tecnologías clave como HTML, CSS, JSON, y JavaScript.

Unidad 1: Introducción a los lenguajes de marcas

Estudiarás las características fundamentales de los lenguajes de marcas como HTML y XML, los cuales permiten estructurar y almacenar información de manera eficiente. Te familiarizarás con las diferentes versiones de estos lenguajes y cómo aplicarlos en contextos reales, analizando la estructura de documentos y comprendiendo las ventajas que ofrecen en el manejo de la información.

Unidad 2: HTML5

Aquí te sumergirás en la construcción de páginas web utilizando HTML5 para la estructura. Validarás el código con herramientas estándar y aplicarás buenas prácticas semánticas para optimizar accesibilidad y SEO.

Unidad 3: CSS3

Verás cómo CSS permite mejorar la presentación visual de las páginas, crear diseños responsive y utilizar características de CSS3 para animaciones, transiciones y otros efectos avanzados.

Unidad 4: Introducción a JavaScript

JavaScript te permitirá interactuar con las páginas web, manipulando el DOM para cambiar dinámicamente contenido, estilos y estructura. Empezarás por los fundamentos del lenguaje y su ejecución en el navegador.

Unidad 4.1: Manipulación del DOM y eventos

Aprenderás a seleccionar, modificar y eliminar elementos, y a responder a eventos del usuario para crear experiencias interactivas y reactivas.

Unidad 5: Definición de esquemas y vocabularios

Trabajarás la descripción y la validación de datos estructurados, empleando convenciones y vocabularios para garantizar la calidad de la información.

Unidad 5.1: JSON Schema y Herramientas de Validación

Aprenderás a validar documentos JSON mediante JSON Schema, asegurando estructura y tipos, y utilizarás herramientas de validación en proyectos reales.

Unidad 5.2: Trabajo con APIs

Entenderás cómo JSON se utiliza para transmitir información entre servidores y clientes, y practicarás peticiones a APIs desde aplicaciones web.

Unidad 5.3: Transformación de documentos con JavaScript

Realizarás transformaciones y formateos de datos con JavaScript para presentar información de forma clara, accesible y reutilizable.

¿Qué vas a aprender?

  1. Lenguajes de Marcas: comprenderás la estructura, versiones y aplicación de HTML y XML en contextos reales.
  2. Creación de Documentos Web: construirás páginas con HTML5 y CSS, validando código y aplicando diseño responsivo.
  3. Programación con JavaScript: manipularás el DOM y gestionarás eventos para crear interfaces interactivas.
  4. Validación de Documentos con JSON: describirás y validarás datos con JSON y JSON Schema, integrándolo en APIs web.
  5. Hojas de Estilo CSS: aplicarás estilos avanzados, animaciones y buenas prácticas de maquetación.

¿Por qué es importante?

El manejo adecuado de estos lenguajes es esencial para el desarrollo web profesional. Aprenderás no solo a crear páginas web estéticamente atractivas, sino también a garantizar que la información sea clara, accesible y bien estructurada. Estas habilidades son clave para desarrollar aplicaciones modernas que interactúan con datos y ofrecen experiencias dinámicas y personalizadas.

Herramientas que usarás

  • HTML5 y CSS3: estructura y estilo.
  • JavaScript: interactividad y lógica en cliente.
  • JSON: intercambio y validación de datos.
  • JSON Schema: definición de estructura y tipos.

Al finalizar el curso, estarás preparado para desarrollar aplicaciones web modernas que gestionen y presenten información de forma eficaz, aplicando buenas prácticas de desarrollo y diseño.
